Resource /api/v1.0.0/icons


Show Icon - Retrieve icons
URL /api/v1.0.0/icons or /api/v1.0.0/icons/:id
Method GET
URL Params

Optional:
id=[type]
example: id=syard:Transformer
Data Params None
Success Response
  • Code: 200
    Content:

    The content returned on success is a PNG icon for the id provided. If no id is provided, then return a list of available icons.
Error Response
  • Code: 401 UNAUTHORIZED
    Content: { error : "You are unauthorized to make this request" }
    • OR
  • Code: 404 NOT FOUND
    Content: { error : "No icon for UUID found" }
Sample Call 1 
	      $.ajax({
	      url: "/api/v1.0.0/icons",
	      dataType: "json",
	      type : "GET",
    	      success : function(r) {
              console.log(r);
	      }
	      });
