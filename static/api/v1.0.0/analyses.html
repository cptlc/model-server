Resource /api/v1.0.0/analyses


Show Analyses - Retrieve analyses
URL /api/v1.0.0/analyses or /api/v1.0.0/analyses/:analysis_id
Method GET
URL Params

Optional:
id=[UUID]
example: id=514d63aa-1fcf-423b-880d-168bac34354f
Data Params None
Success Response
  • Code: 200
    Content: { 'nodes': [...], 'links':[...] }

    The content returned on success should validate against the schema provided.
Error Response
  • Code: 401 UNAUTHORIZED
    Content: { error : "You are unauthorized to make this request" }
    • OR
  • Code: 404 NOT FOUND
    Content: { error : "No graph for UUID found" }
Sample Call 1 
	      $.ajax({
	      url: "/api/v1.0.0/analyses",
	      dataType: "json",
	      type : "GET",
    	      success : function(r) {
              console.log(r);
	      }
	      });
